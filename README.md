# 微信小程序-天气
天气接口用的是百度接口 

基本功能就是可以获取近四天的天气情况

也可以 通过输入城市名称获取

使用方法 :

```
appid是我自己的 可以填写自己的appid就可以运行了
```

很简单的一个小程序 没啥东西!
